API Ayarları

OpenAI ve WhatsApp Business API bağlantılarını yapılandırın.

beyin.py Konfigürasyonu

Bot altyapınız için Python dosyasında yapmanız gerekenler:

1. client = OpenAI(api_key="sk-...")
   → API anahtarınızı girin

2. model="gpt-4o-mini"
   → Seçtiğiniz modeli yazın

3. json_dosya_yolu
   → veri.json ile aynı klasörde olsun

4. Sipariş formatı:
   [SIPARIS: Ad | Tel | Adres | Ürün]
   → Bu format değiştirilmemeli

5. WhatsApp bağlantısı için:
   → Meta Business API entegrasyonu
   → Phone Number ID ekleyin
   → Webhook kurun
